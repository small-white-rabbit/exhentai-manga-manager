# 轻量漫画阅读器 (Manga Reader)

基于 Rust 和 `egui` 开发的本地桌面跨平台漫画阅读器。

## 开发目的

开发此应用是为了最小化实现一个轻量的漫画阅读器，仅保留最必要的功能和最小设置，作为exhentai-manga-manager的默认外置阅读器使用。

## 核心特性

* **阅读模式**：
  * **滚动模式 (Scroll)**：垂直滚动页面浏览。
  * **单页模式 (Single Page)**：单页自适应屏幕缩放浏览。
  * **双页模式 (Double Page)**：以固定的从右到左排版一次显示两张图片并无缝拼接，同时支持切换“第一页单独显示”。
* **格式支持**：
  * 图像格式：`JPG`, `JPEG`, `PNG`, `WEBP`, `GIF`。
  * 压缩包：读取并渲染 `ZIP/CBZ`, `RAR/CBR`, `7Z/CB7` 格式的压缩文件。
* **内存管理**：
  * **内存提取**：读取压缩包时在内存中提取图像（当漫画文件大小超过 1GB 阈值时回退至临时目录解压）。
  * **缓存清理**：滑动加载图像，同时清理视野外的图像缓存以控制内存占用。
* **参数调用**：支持通过命令行参数传入文件夹或压缩包路径打开。

## 快捷键与操作说明

### 卷轴模式 (Scroll Mode)
* **鼠标滚轮**：垂直滚动页面。
* **`↑` / `↓` 方向键**：平滑垂直滚动。
* **`←` / `→` 方向键**：按屏幕高度翻页滚动（受工具栏 `RTL` 开关影响）。
* **`Space`**：下一页滚动。

### 单页/双页模式 (Single/Double Page Mode)
* **鼠标点击**：点击窗口左/右半侧翻页或翻页组（受工具栏 `RTL` 开关影响）。
* **鼠标滚轮**：向上滚动上一页，向下滚动下一页。
* **`←` / `→` 方向键**：翻页（受工具栏 `RTL` 开关影响）。
* **`Space`**：下一页。
* **`/` (斜杠)**：仅双页模式下生效，用于切换**第一页是否仅显示一张图片**。

### 通用操作
* **`Esc` / 鼠标后退键 (Back)**：关闭阅读器窗口。
* **拖拽文件夹或压缩包到窗口**：直接加载并开始阅读。

## 编译与运行

确保你已经准备好了 Rust 开发环境。

```bash
# 运行
cargo run

# 编译
cargo build --release
```
